# Awesome-Vivaldi

修改自 https://github.com/PaRr0tBoY/Awesome-Vivaldi
捨棄js僅css，不包括多餘功能，僅測試於 Windows 環境 (Win11-24H2)

## 設定步驟
- `Vivaldi://experiments` 開啟 `Allow CSS Modification` 功能，在自訂CSS設定中選擇本倉庫的 `CSS` 資料夾
- 垂直標籤欄 (左右皆可)
- 浮動面版
- 顯示標題列
- 其餘依照喜好

## 主要功能
- 自動收束垂直標籤欄
- 縮小標題列使之可以顯示且不會占太多空間
- 自動隱藏面板欄
- 自動隱藏底部工具列
